# Ateliers.Voice.Engines.VoicevoxTools

VOICEVOX Core を使用した音声生成エンジン。

## 概要

Ateliers.Voice.Engines.VoicevoxTools は、VOICEVOX Core（VoicevoxCoreSharp）をラップし、C#から簡単に使用できる音声生成APIを提供します。

## 主な機能

- **VoicevoxVoiceGenerator**: VOICEVOX音声生成の中核クラス
- **VoicevoxGenerationOptions**: 音声生成オプション（スタイル、速度、ピッチなど）
- **VoicevoxMetadata**: 生成された音声のメタデータ管理

## 依存関係

- Ateliers.Voice.Engines
- Ateliers.Core
- VoicevoxCoreSharp.Core
- Microsoft.Extensions.FileSystemGlobbing
