# Ateliers.Voice.Engines

音声エンジンの共通ユーティリティとヘルパークラスを提供するライブラリ。

## 概要

Ateliers.Voice.Engines は、音声生成エンジン（VOICEVOX、VOICEPEAKなど）で共通して使用されるユーティリティやヘルパークラスを提供します。

## 主な機能

- **VoiceOutputDirectoryHelper**: タイムスタンプベースのディレクトリ生成とファイルパス構築
- **TextFileSaveMode**: 音声生成時のテキストファイル保存モード定義

## 依存関係

なし（.NET 10 標準ライブラリのみ）
